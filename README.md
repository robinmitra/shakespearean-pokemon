# Shakespearean Pokemon

A minimalist API that takes a Pokemon species and returns its Shakespearean description!

## Usage

```shell script
docker-compose up --build
```

## Development

### Run without building

```shell script
go run main.go
```

### Build and run

```shell script
go build
```

You can then run the generated binary directly.

### Run the tests

```shell script
go test ./...
```
